# Clash95 Ghidra Export

Generated by `reverse/ghidra/ExportClash95Facts.java` from `clash95.exe` and `clash95.map`.

Outputs:
- `metadata.txt`: PE/program metadata and memory blocks.
- `imports.csv`: imported external symbols grouped by library.
- `functions.csv`: Ghidra function inventory after applying map names.
- `selected_decompilation.c`: Ghidra decompiler output for first-pass port targets.
